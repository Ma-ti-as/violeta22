Flexbox y cómo se utiliza

"Flexible box layout": Solución para desarrollos web responsive compatible con todos los navegadores actuales y que se añade a la lista de displays y posicionamientos conocidos hasta 2018:

  • Elementos en linea (display:inline)
  • Elementos en bloque (display:block)
  • Elementos en tabla (display:table)
  • Tipos de posicionamiento (position:absolute, relative,fixed,stiky,..)
  • Layout flexible (display:flex)
  • .....

Display:flex decidir el comportamiento de los hijos de un contenedor padre con respecto a el y entre ellos: Tanto en la vertical cómo en la horizontal (Híbrido de los otros métodos anteriores de posicionamiento)

W3C define esta estructura como flujo flexible de contenidos siempre de izquierda a derecha (eje horizontal o principal) y de arriba a abajo (eje vertical o secundario

Ejemplos de maquetación con Flexbox, configuración desde los elementos o etiquetas hijo del contenedor

Las propiedades del elemento flexible son:

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

La propiedad order, especifica el orden de los elementos flexibles.
El primer elemento flexible del código no tiene que aparecer como el primer elemento del diseño.
El valor debe ser un número y su valor predeterminado es 0. Por tanto un número mayor desplaza este elemento al final del contenedor.

La propiedad grow especifica la capacidad de crecimiento de los elementos flexibles.
El valor debe ser un número y su valor predeterminado es 0. Por tanto un valor = 1 en un elemento flexible hace que este crezca el doble que sus hermanos
